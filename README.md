
### Crusader Map Generator

Procedural map generator for Stronghold Crusader 1

#### Installation and usage

The program can only run on Windows, both 64bit and 32bit builds are available

It requires the game to operate

It is only compatible with v1.41 of the game (latest Steam/GoG release, or patched) in English (and most latin-alphabet languages)

Other game versions and localizations are not supported, Extreme is not supported

1. Download the latest release
Extract the files, place them all in the same folder anywhere on your PC
2. Set up your config
Do this by editing the "config.txt" file in the program's directory
A default config is provided
It is verified before each run with arbitrary but very lenient restrictions, it is still very easy to set impossible or otherwise stupid values
You may reorder the config variables however you want, but every variable must exist and have a valid value
Explaining all the variables, their function and proper values fully would be too much work, change things and see how it affects the output
3. Run the game
If you have automatic map saving enabled, you can leave it in the main menu, otherwise create a new map in the map editor
4. Run the program
This requires admin permissions
When you run the generator, make sure to wait before it's finished before interacting with the game, otherwise the output may be corrupted and/or the game may crash
If your antivirus quarantines the program, create an exception for it and restore or reinstall it
